Mis Proyectos

Desafio #01 - Curriculum Vitae en HTML

Primer desafío que pone en práctica los conocimientos adquiridos en HTML para crear una maqueta de un Currículum Vitae, basada en la trayectoria y los logros del futbolista portugués Cristiano Ronaldo

Desafio #02 - Construyendo un landing page

Segundo desafío que pone en práctica los conocimientos adquiridos en HTML más CSS para crear una landing page simple basada en la empresa Decohogar con un minicatálogo de plantas con sus precios.

Desafio #03 - Iguana page

Tercer desafío que pone en práctica los conocimientos adquiridos en HTML más CSS y Flexbox para crear una pagina con varias secciones basada en las iguanas, respetando una estructua de DOM orientada a páginas web reales.

Desafío #04 - Cuppon

Cuarto desafío que pone en práctica los conocimientos adquiridos en HTML más CSS, Flexbox y Framkework Bootstrap para crear una pagina de cupones que cuenta con etiquetas card que muestran un catálogo de productos o experiencias.

Desafío #05 - Github

Quinto desafío que pone en práctica los conocimientos adquiridos en HTML más CSS, Flexbox, Framkework Bootstrap y Git para trabajo colaborativo, modificando secciones de una página y realizando correcciones colaborativas.
